# 💳 Credit Card Fraud Risk Analysis

## 📌 Project Overview

This project focuses on analyzing credit card transactions to identify fraudulent activities using data visualization and risk analysis techniques in **Power BI**.

The dashboard helps in:

* Detecting suspicious transaction patterns
* Monitoring fraud trends
* Understanding customer transaction behavior
* Supporting fraud prevention decision-making

---

## 🚀 Features

* 📊 Interactive Power BI Dashboard
* 📈 Fraud vs Non-Fraud Transaction Analysis
* 🌍 Region-wise Fraud Detection
* 💰 Transaction Amount Analysis
* ⏰ Time-based Fraud Trends
* 🔍 Risk Insights using Visual Analytics
* 📌 KPI Cards for Quick Monitoring

---

## 🛠️ Tech Stack

* **Power BI**
* **Data Analytics**
* **Data Visualization**
* **Excel / CSV Dataset**
* **DAX (Data Analysis Expressions)**

## 📊 Key Insights

* Fraudulent transactions occur more frequently during unusual transaction hours.
* High-value transactions show increased fraud probability.
* Certain regions/customers have higher fraud risk patterns.
* Real-time monitoring can significantly reduce financial losses.
